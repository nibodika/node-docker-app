# 🚀 Node.js Docker App

A simple Node.js application containerized using Docker.

## 🛠 Features
- Basic Node.js HTTP server
- Dockerized application
- Runs on port 3000

## 🐳 Run with Docker

### Build Image
docker build -t node-app:v1 .

### Run Container
docker run -d -p 3000:3000 node-app:v1

## 🌐 Access
http://localhost:3000
